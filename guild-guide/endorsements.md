# Guild Endorsements

Endorsements are how your guild amplifies the voices and ideas that matter most. When your guild endorses a CivicNote, proposal, or petition, you're signaling to the community—and to officials—that this issue has organized support behind it. This guide explains the endorsement process, from suggesting a note to celebrating its impact.

---

## What Is an Endorsement?

An **endorsement** is your guild's official stamp of approval on a piece of content. Endorsed items receive:

- A **guild badge** visible on the item (e.g., "🛡️ Safe Streets Guild")
- **Increased visibility** in feeds and search results
- **Higher weighting** in some platform metrics (e.g., Promise Tracker verifications)
- **Tracking** in your guild's analytics and impact reports

Endorsements tell the world: "This matters to us, and we stand behind it."

### What Can Be Endorsed?

| Content Type | What Endorsement Means |
|--------------|------------------------|
| **CivicNote** | Your guild agrees with the sentiment or proposal expressed. |
| **Proposal** | Your guild officially supports this formal request. |
| **Petition** | Your guild encourages members and the public to sign. |
| **Promise** | (Future feature) Your guild verifies this promise's accuracy. |

---

## The Endorsement Lifecycle
```markdown
Suggest → Queue → Discuss → Vote → Approve/Decline → (if approved) Badge Added → Track Impact
```

| Stage | Description |
|-------|-------------|
| **Suggest** | A member finds an item and flags it for consideration. |
| **Queue** | The item appears in the guild's endorsement queue. |
| **Discuss** | Members discuss the item in the attached thread. |
| **Vote** | Endorsement managers (or all members) vote on approval. |
| **Approve/Decline** | If approved, the item receives the guild badge. If declined, it's removed from queue. |
| **Track Impact** | The endorsed item is tracked in guild analytics and Promise Tracker. |

## Step 1: Suggesting an Endorsement

Any guild member can suggest an endorsement. This ensures the whole guild has a voice in what gets supported.

### How to Suggest

1. Find a CivicNote, proposal, or petition you believe your guild should endorse.
2. Look for the **"Suggest Guild Endorsement"** button (usually in the item's action menu or below the content).
```markdown
+------------------------------------------------------------------+
| Alex · District 3 · Apr 12, 2025 · [Proposal 💡] |
| |
| This intersection is already dangerous for kids walking to |
| school. Rezoning without a traffic calming plan is reckless. |
| |
| [👍 12] [💬 3] [🔗 2] [Suggest Guild Endorsement] |
+------------------------------------------------------------------+
```

3. Click the button. A brief form may appear asking for optional comments (why you think it deserves endorsement).
4. Submit. The item now appears in your guild's **Endorsement Queue**.

### What Happens Next

- The suggester receives a confirmation.
- All guild members with notification settings enabled receive an alert: "New endorsement suggestion from [name]."
- The item enters the queue for discussion and voting.

---

## Step 2: The Endorsement Queue

The queue lives in your guild dashboard under **"Proposals & Endorsements"** (or a dedicated "Endorsements" tab). It shows all items currently under consideration.
```markdown
+------------------------------------------------------------------+
| ⭐ ENDORSEMENT QUEUE (3) |
| +----------------------------------------------------------------+
| | Item | Suggested By | Votes | Deadline |
| |-------------------------|--------------|---------|-------------|
| | 5th & Main crosswalk | Alex | 5/10 | 2 days left |
| | School zone speed | Jamie | 3/10 | 4 days left |
| | Bike lane repair | Priya | 8/10 | 1 day left |
| +----------------------------------------------------------------+
| |
| [View All] [Filter by Topic] [Sort by Deadline] |
+------------------------------------------------------------------+
```

From here, members can:

- **Click any item** to see full details, including the original content and the suggester's comments.
- **Join the discussion** in the attached thread.
- **Cast a vote** (if your guild uses voting for endorsements).

## Step 3: Discussing an Endorsement

Every endorsement suggestion has a dedicated discussion thread, visible only to guild members.
```markdown
+------------------------------------------------------------------+
| DISCUSSION: 5th & Main crosswalk note by Alex |
+------------------------------------------------------------------+
| |
| Alex (suggester): "This note captures exactly what we've been |
| talking about. The crosswalk is dangerous and the proposal for |
| rapid-flashing beacons is concrete." |
| |
| Jamie: "I agree. But should we also ask for a full traffic study?|
| Maybe we should endorse with a comment suggesting that addition."|
| |
| Priya: "Good point. I'll add a note to the discussion. We can |
| endorse and then follow up with a separate proposal." |
| |
| Taylor: "👍" |
| |
| [Add Comment] |
+------------------------------------------------------------------+
```

Discussions help refine the guild's position and ensure everyone is informed before voting.

## Step 4: Voting

Endorsement voting rules are set in your guild's **Settings**. Common models:

| Model | Description |
|-------|-------------|
| **Simple majority** | More than 50% of votes cast approves. |
| **Supermajority (e.g., 60%)** | Requires higher consensus, good for controversial items. |
| **Consensus** | No formal vote; discussion continues until no objections remain. |
| **Endorsement manager decision** | A designated member or team decides after hearing discussion. |

### How to Vote

When a vote is active, eligible members see voting buttons on the item's detail page:
```markdown
+------------------------------------------------------------------+
| CAST YOUR VOTE |
| |
| Should the Safe Streets Guild endorse this note? |
| |
| ○ Yes, endorse |
| ○ No, do not endorse |
| ○ Abstain |
| |
| Optional comment: [_____________________________] |
| |
| [Submit Vote] |
+------------------------------------------------------------------+
```

Votes are visible to guild members (but not to the public) to ensure transparency.

### Voting Thresholds

The endorsement becomes official when:

- **Minimum votes cast** – Your guild may set a minimum participation (e.g., at least 10 votes) to ensure decisions aren't made by a handful of members.
- **Threshold percentage** – e.g., 60% of votes cast must be "Yes."

If the threshold isn't met within the voting period (e.g., 7 days), the item may expire or return to discussion.

---

## Step 5: Approval & Badge Placement

Once approved, the endorsement becomes official. The system automatically:

1. Adds a **guild badge** to the endorsed item:
```markdown
+------------------------------------------------------------------+
| Alex · District 3 · Apr 12, 2025 · [Proposal 💡] |
| |
| [🛡️ Safe Streets Guild] |
| |
| This intersection is already dangerous for kids walking to |
| school... |
+------------------------------------------------------------------+
```

2. **Notifies** the original author (if a CivicNote or proposal) that their item was endorsed.
3. **Updates guild analytics** – the endorsed item is now tracked in your guild's impact metrics.
4. **Adds to endorsed items list** in your guild profile (public) and dashboard (private).

The badge links to your guild's profile, allowing anyone to learn more about who endorses this content.

---

## Step 6: Tracking Impact

Endorsed items are tracked in your guild's **Analytics** tab. You can see:

- How many endorsed notes were viewed, liked, or replied to.
- Which endorsed proposals led to promises or official acknowledgment.
- Petition signature growth after endorsement.
- Overall Φ‑score contribution from endorsed items.

This data helps your guild understand what's working and where to focus future efforts.

## Best Practices for Endorsements

### Do's

- **Endorse strategically** – Quality over quantity. Each endorsement should reflect genuine guild consensus.
- **Discuss before voting** – Surface diverse perspectives; stronger decisions emerge from dialogue.
- **Celebrate endorsed items** – Share them in announcements and encourage members to engage.
- **Follow up** – Track whether endorsed items lead to action (promises, petitions). Close the loop.

### Don'ts

- **Endorse without discussion** – Especially for controversial items, skip the discussion at your peril.
- **Endorse everything** – Dilutes the meaning of your badge.
- **Ignore the queue** – Endorsement suggestions are gifts from members; acknowledge them.
- **Forget to update** – If a proposal changes significantly, consider whether endorsement should be reconsidered.

---

## Revoking an Endorsement

Sometimes circumstances change – a proposal may be amended, or new information emerges. Guilds can revoke endorsements.

### How to Revoke

1. Any member can suggest revocation via the same process as suggesting endorsement.
2. The item appears in the queue with a "Revocation" flag.
3. Discussion and vote follow the same rules as endorsement approval.
4. If revocation passes, the guild badge is removed from the item.
5. A public note may be added explaining the change (optional but recommended).

Revocation should be used sparingly and with clear communication.

## Frequently Asked Questions

### Can a note be endorsed by multiple guilds?
Yes. A note may show badges from several guilds, indicating broad coalition support.

### Who can see that I voted?
Votes are visible only within the guild (to members). The public never sees individual votes, only the final endorsement decision.

### What if I disagree with an endorsement?
Participate in the discussion and vote your conscience. If the decision goes against your view, respect the collective process. You can also suggest revocation if new information emerges.

### How do we handle urgent endorsements (e.g., a last‑minute proposal)?
Some guilds set a "fast‑track" rule for urgent items (e.g., 24‑hour voting period, or delegation to endorsement managers). Define this in your charter or guild settings.

### Can we endorse our own guild's proposals?
Yes, but it's good practice to have a different member suggest the endorsement to ensure genuine collective support.

### Do endorsements expire?
Not automatically, but guilds can periodically review their endorsed items list and revoke if priorities have shifted.
