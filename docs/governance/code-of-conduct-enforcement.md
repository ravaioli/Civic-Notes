# Code of Conduct Enforcement

The Code of Conduct is the foundation of a safe, respectful, and productive community on CivicNotes. But a code without enforcement is merely a suggestion. This guide explains how the Code is enforced—how reports are filed, how investigations are conducted, how sanctions are determined, and how the community ensures accountability at every step.

---

## Why Enforcement Matters

Effective enforcement:

- **Protects individuals** from harassment, abuse, and harm.
- **Preserves community trust** that rules will be applied fairly.
- **Deters misconduct** by demonstrating consequences.
- **Upholds platform values** of respect, transparency, and regeneration.
- **Provides recourse** for those who have been harmed.

Enforcement is not about punishment for its own sake—it's about maintaining the conditions for healthy civic engagement.

---

## Enforcement Principles

| Principle | Application |
|-----------|-------------|
| **Fairness** | All parties are heard; decisions are based on evidence. |
| **Proportionality** | Sanctions match the severity and impact of the violation. |
| **Restorative focus** | Where possible, seek to repair harm and restore relationships. |
| **Transparency** | Processes are clear; decisions are explained (with appropriate confidentiality). |
| **Consistency** | Similar cases yield similar outcomes. |
| **Appealability** | All decisions can be challenged through defined processes. |
| **Continuous improvement** | Lessons learned inform updates to the Code and processes. |

---

## Who Enforces the Code?

Enforcement is shared across different levels of the community:

| Level | Responsible | Scope |
|-------|-------------|-------|
| **Individual users** | Anyone can report violations. | Reporting |
| **Guilds** | Guild leaders may address internal violations (e.g., member conduct). | Within guild |
| **Moderation Council** | Primary enforcement body for platform‑wide violations. | All Code violations |
| **Platform administrators** | Technical enforcement (e.g., account suspension) based on Council decisions. | Implementation |
| **Guild Council** | Appellate body for Council decisions. | Appeals |

The **Moderation Council** is the primary judicial body for Code enforcement. See [Moderation Council](moderation-council.md) for details on its composition and powers.

## What Constitutes a Code Violation?

The [Code of Conduct](../CODE_OF_CONDUCT.md) prohibits:

| Category | Examples |
|----------|----------|
| **Harassment** | Targeted attacks, intimidation, stalking, unwanted sexual attention. |
| **Hate speech** | Content that promotes violence or hatred based on identity. |
| **Threats** | Statements threatening violence or harm. |
| **Impersonation** | Pretending to be someone else (especially officials or other users). |
| **Fraud** | Fake signatures, false evidence, deceptive practices. |
| **Privacy violations** | Sharing others' personal information without consent. |
| **Spam** | Repetitive, unsolicited, or disruptive content. |
| **Evasion** | Creating new accounts to bypass sanctions. |
| **Disruption** | Deliberately interfering with platform functions or community processes. |

This list is not exhaustive. The Moderation Council interprets the Code in context.

## Reporting a Violation

### How to Report

| Method | Where | Best For |
|--------|-------|----------|
| **Report button** | On any note, comment, profile, or petition. | Quick reporting of specific content. |
| **Request Moderation form** | Help section or "Report" link. | Complex situations, ongoing behavior, need for investigation. |
| **Direct message to Council** | Via Moderation Council contact (moderation@civicnotes.org). | Sensitive matters, or if other methods aren't working. |

### What to Include

When reporting, provide:

- **What happened** – Clear description of the incident(s).
- **Who was involved** – Usernames of those involved.
- **Where** – Links to specific content (notes, comments, etc.).
- **When** – Dates and times (if known).
- **Evidence** – Screenshots, links, messages (preserve originals).
- **Impact** – How you or others were affected.
- **Prior attempts** – Have you tried addressing it directly? (Not required for serious violations.)

### Anonymous Reporting

You may report anonymously using the form, but anonymous reports may be harder to investigate. If you want to participate in the process, you'll need to identify yourself.

## Enforcement Process Overview
```markdown
Report Filed → Triage → Investigation → Determination → Sanction (if any) → (Optional) Appeal
```

| Stage | Description | Timeline |
|-------|-------------|----------|
| **Report filed** | User submits report via appropriate channel. | Immediate |
| **Triage** | Council member reviews for urgency and jurisdiction. | Within 24–48 hours |
| **Investigation** | Primary reviewer gathers facts, interviews parties. | 3–14 days |
| **Determination** | Full Council reviews and decides. | 1–7 days after investigation |
| **Sanction imposed** | If violation found, sanction is applied. | Within 24 hours of decision |
| **Appeal** | Party may appeal to Guild Council. | Within 14 days |

---

## Step 1: Triage

A designated Council member (rotating duty) reviews each report within **24–48 hours**.

### Triage Questions

- Is this within the Council's jurisdiction? (Platform matters, not external disputes.)
- Is there sufficient information to proceed?
- Is there immediate harm requiring urgent action?
- Have lower‑level remedies been attempted (if appropriate)?

### Possible Triage Outcomes

| Outcome | Description |
|---------|-------------|
| **Accept** | Case proceeds to investigation. |
| **Reject** | Not a violation, insufficient info, or wrong jurisdiction. Filer receives explanation. |
| **Refer** | Better handled by guild or other body (e.g., internal guild dispute). |
| **Urgent action** | Temporary measures (content removal, account suspension) taken pending full review. |

---

## Step 2: Investigation

The case is assigned to a **primary reviewer** (one Council member).

### Investigator's Role

- Gather all relevant evidence.
- Contact the reporter for clarification if needed.
- Contact the accused user (respondent) to inform them of the report and give opportunity to respond.
- Interview witnesses if applicable.
- Maintain confidentiality.

### Respondent's Rights

- To be informed of the report (with sufficient detail to respond).
- To provide their perspective and evidence.
- To have support person present during any meetings.
- To know the status of the investigation.

### Investigation Timeline

- Most investigations complete within **7–14 days**.
- Complex cases may take longer; parties are updated.

---

## Step 3: Determination

Primary reviewer presents findings to the full Moderation Council. Council discusses and votes.

### Questions the Council Considers

- **Did the conduct occur?** (Factual determination based on evidence.)
- **Does it violate the Code?** (Interpretation of rules in context.)
- **What was the impact?** (Harm to individuals, community, platform.)
- **Were there mitigating factors?** (Remorse, good faith, context.)
- **Were there aggravating factors?** (Pattern of behavior, intentionality.)

### Decision Options

| Outcome | Description |
|---------|-------------|
| **No violation** | Conduct does not violate Code; case closed. |
| **Violation – sanction** | Conduct violated Code; sanction applied. |
| **Violation – restorative resolution** | Conduct violated Code but restorative measures (e.g., apology, education) may be sufficient. |
| **Inconclusive** | Insufficient evidence; case closed with note. |

### Decision Documentation

Each decision includes:

- Summary of findings.
- Interpretation of relevant Code provisions.
- Rationale for outcome.
- Sanction (if any).
- Appeal rights.

Decisions are published in the **Case Log** (with personal information redacted).

---

## Step 4: Sanctions

Sanctions are graduated and proportional to the violation.

### Sanction Levels

| Level | Sanction | Typical Duration | Examples |
|-------|----------|------------------|----------|
| **1** | Warning | Permanent record | First offense, minor impact, good faith. |
| **2** | Feature restriction | 30–90 days | Cannot propose promises, endorse, or create petitions. |
| **3** | Temporary suspension | 7–30 days | Cannot post, comment, or interact. |
| **4** | Extended suspension | 31–180 days | Longer restriction for serious violations. |
| **5** | Permanent ban | Indefinite | Severe or repeated violations, threats, fraud. |
| **6** | Restorative measures | Varies | Apology, education, mediation – may accompany other sanctions. |

### Factors Influencing Sanction

| Factor | Effect |
|--------|--------|
| **First offense** | Lower sanction likely |
| **Pattern of behavior** | Higher sanction |
| **Intentional vs. accidental** | Intentional more severe |
| **Impact on victim** | Greater harm, higher sanction |
| **Remorse/accountability** | May mitigate |
| **Cooperation with investigation** | May mitigate |
| **Attempts to repair harm** | May mitigate |

### Restorative Measures

Where appropriate, the Council may recommend restorative actions:

- **Apology** – Public or private, as agreed.
- **Education** – Complete a module on respectful communication.
- **Mediation** – Participate in facilitated conversation with harmed party.
- **Restorative circle** – Community process to address harm.

Restorative measures do not replace sanctions but may accompany them or, in minor cases, substitute for punitive action.

---

## Step 5: Implementation

Once the Council issues a decision:

- **Platform administrators** implement technical sanctions (suspensions, bans, feature restrictions).
- **The user** is notified of the decision and sanction.
- **The reporter** is notified of the outcome (with appropriate confidentiality).
- **Case Log** updated with redacted decision.

### Timing

Sanctions are implemented within **24 hours** of decision.

---

## Step 6: Appeals

Any party (reporter or respondent) may appeal a Council decision.

### Grounds for Appeal

- **Procedural error** – Council failed to follow its own processes.
- **New evidence** – Material evidence not previously available.
- **Misinterpretation** – Council incorrectly interpreted the Code.
- **Disproportionality** – Sanction clearly disproportionate to violation.

### Appeal Process

1. File appeal within **14 days** of decision.
2. Appeal must specify grounds and include supporting information.
3. Guild Council reviews at next regular meeting.
4. Guild Council may:
   - **Uphold** Council decision.
   - **Remand** to Council for reconsideration with guidance.
   - **Reverse** decision (modify or vacate sanction).

Guild Council decisions are final.

## Special Situations

### Urgent Threats

If a user poses an immediate threat to safety (e.g., violent rhetoric, doxxing), the Council may:

- Temporarily suspend the account pending full review.
- Remove the threatening content immediately.
- Report to appropriate authorities if necessary.

### Evasion of Sanctions

Creating new accounts to bypass a suspension or ban is a separate violation and may result in:

- Immediate permanent ban of all accounts.
- Extended ban duration for original account.

### False Reports

Knowingly filing false reports is a Code violation and may result in sanctions against the reporter.

### Appeals of Admin Actions

If a user believes platform administrators acted improperly (e.g., unjust suspension), they may appeal to the Moderation Council. The Council reviews and may uphold, overturn, or modify the admin action.

## Transparency and Confidentiality

### What Is Public

- Final decisions (redacted) in Case Log.
- Aggregate statistics in annual report.
- Sanctions applied (anonymized) for certain cases (e.g., permanent bans are noted publicly).

### What Is Confidential

- Identity of parties (unless public figure and relevant).
- Testimony and evidence.
- Deliberations.

### Exceptions

Confidentiality may be waived:
- With consent of all parties.
- When necessary to protect the community (e.g., warning about a banned user).
- When required by law.

## User Rights Throughout Process

| Right | Description |
|-------|-------------|
| **To be informed** | Know that a report has been filed (if you're the respondent). |
| **To respond** | Provide your perspective and evidence. |
| **To have support** | Bring a supporter to meetings. |
| **To know the outcome** | Receive decision with rationale. |
| **To appeal** | Challenge decisions through defined process. |
| **To confidentiality** | Within limits above. |
| **To be free from retaliation** | No adverse action for filing a good‑faith report. |

---

## Retaliation Prohibited

Retaliation against anyone who files a report, participates in an investigation, or exercises their rights under the Code is itself a Code violation and may result in sanctions.

If you experience retaliation, report it immediately.

## Enforcement Statistics and Reporting

The Moderation Council publishes an **annual report** including:

- Number of reports filed.
- Types of violations.
- Outcomes (sanctions issued, restorative resolutions).
- Average response times.
- Trends and observations.
- Recommendations for Code or process improvements.

This transparency builds trust and enables continuous improvement.

## Frequently Asked Questions

### How do I report someone?
Use the "Report" button on their profile or content, or the "Request Moderation" form.

### What if I'm not sure it's a violation?
Report it anyway. The Council will determine if it violates the Code.

### How long does the process take?
Most cases resolve within 2–3 weeks. Complex cases may take longer.

### Will I know who reported me?
Not necessarily. The Council protects reporter confidentiality where possible, unless disclosure is necessary for the investigation.

### Can I be sanctioned for something I said in a private message?
Yes, if it constitutes harassment, threats, or other Code violations. The Code applies to all platform interactions, including DMs.

### What if the person harassing me is also in my guild?
Report to the Council. Guilds have their own processes, but the Council can intervene for Code violations.

### Can I appeal if I think the sanction is too harsh?
Yes, within 14 days. See appeals process above.

### What happens to my account during an investigation?
Generally, no action until a decision is made, unless there's urgent risk requiring temporary suspension.

### How are permanent bans enforced?
Banned users cannot create new accounts; evasion attempts result in additional sanctions.

### Can a banned user ever return?
Permanent bans are rare and reserved for the most severe cases. There is no automatic reinstatement, but a banned user may petition the Council after a significant period (e.g., 2 years) demonstrating changed circumstances.
